# 🖥️ codyssey_imac

교육장 iMac 초기화 환경에서 로그아웃 없이 마우스, 키보드, 언어 설정을 즉시 반영하기 위한 자동화 스크립트 저장소입니다.

## 🚀 한 줄 실행 명령어

교육장 자리에 앉자마자 **터미널(Terminal)**을 열고 아래 명령어를 복사해서 엔터만 치면 세팅이 끝납니다.

```bash
curl -sL [https://raw.githubusercontent.com/cody-hong/codyssey_imac/main/setup.sh](https://raw.githubusercontent.com/cody-hong/codyssey_imac/main/setup.sh) | bash
